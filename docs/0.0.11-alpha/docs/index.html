インラインテキスト要素

マーク要素: mark要素でテキストをハイライト表示する。

del要素: 削除したことを示す要素。

s要素: もはや関連性がないことを表すために取り消し線を引きたいときに使う。

ins要素: 追加したテキストであることを示す要素。

u要素: テキストに下線を引く要素。

small要素: テキストを、親要素の86%のフォント・サイズで表示する。

strong要素: 強調したいテキストを、太字で表示する。

em要素: テキストをイタリック体で表示する。
